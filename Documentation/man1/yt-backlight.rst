yt-backlight

Backlight control
-----------------

:Manual section: 1
:Date: 2018-06-16
:Author: Jan MatÄ›jka jan@matejka.ninja
:Manual group: yac tools manual

SYNOPSIS
========

::

  yt backlight [--max]
  yt backlight <new>

DESCRIPTION
===========

Without arguments prints actual brightness of the backlight. With <new> sets the backlight to the
<new>.

OPTIONS
=======

--max     Prints max brightness instead of actual one.
